class CheckBit
{
    static boolean checkKthBit(int n, int k)
    {
        if(1==((n>>k)&1)){
            return true;
        }
        else{
            return false;
        }
    }
    
}


test case:

Input: 
N = 4
K = 0
Output: 
No
Explanation: 
Binary representation of 4 is 100, in which 0th index bit from LSB is not set. So, return false.
